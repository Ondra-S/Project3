## Project Overview

This is a static web project consisting of `index.html` and `style.css`. There is no build system, package manager, or test framework configured yet.

## Development

Open `index.html` directly in a browser, or use a local static file server:

```bash
# Python
python -m http.server 8080

# Node.js (if npx available)
npx serve .
```
